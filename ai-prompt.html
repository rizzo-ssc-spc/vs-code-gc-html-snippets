Create an HTML snippet collection to be used on VS Code with all of the HTML components from the Government of Canada. This snippet collection will include various components such as, but not limited to, headers, footers, navigation menus, buttons, forms, tables, panels, cards, etc. Each snippet will be designed to follow the Government of Canada Web Standards (GCWeb) and the Government of Canada Design System (GCDS) guidelines. It will be share with the Web publishing team to ensure consistency and improve productivity. Use WET, GCWeb and GC Design System components (e.g.,wet-footer,gcds-header). Ensure GCDS typography, color contrast, and spacing. Include WCAG 2.1 AA accessibility features (ARIA labels). Scrape the entire library from the following resources provided: https://wet-boew.github.io/wet-boew-styleguide/index-en.html https://design-system.canada.ca/en/ https://wet-boew.github.io/GCWeb/index-en.html Output as a valid VS Code JSON format. Organized into logical categories, add smart placeholders for dynamic content and configure for HTML scope (works in .html, .njk, .liquid, .vue, .jsx). Please also create a documentation for quick reference and how-to-guide. Assume that not all users will be familiar with how to install and use snippets in VS Code.
